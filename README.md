
# Weather-app

A weather app using an API and React js.



## Authors

- [@Georgeb779](https://github.com/Georgeb779)

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/Georgeb779/Weather-app.git
```

Go to the project directory

```bash
  cd Weather-app
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev 

```
Create a build

```bash
npm run build 
```
